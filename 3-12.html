Second Visualization

This visualization does not present the whole dataset at once, which makes it more striking overall. There are multiple pieces of data presented on the same svg at once for the different information shown. What I like about the visualization is that it tells a story, and has a concluding ending or answer.

The effects include using a hover effect that displays the year associated with the data point the mouse is hovering over. There may also be a transition utilized that delays the data presentation so the viewer understands how the data changes over time, which is what is most important to their story. What triggers these animations, however, is not a mouseover effect but it seems to be a scroll effect, which is interesting. Depending on where the user is located on the page triggers which animation to start. I think a way to implement it based on what we know is to create an invisible rectangle that takes up the whole width of the page but not the whole height and on mouseover, trigger the animation, and for other animations do the same.
